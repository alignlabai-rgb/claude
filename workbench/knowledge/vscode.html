VS Code 基本ガイド（プレビュー）

エディタの開き方、ファイル移動、コマンドパレット、ターミナル、マルチカーソルなど、最初に覚えると楽になる操作をまとめました。随時拡充します。

← Workbenchへ戻る
最初に

最低限これだけ

  • ファイルを開く: Cmd/Ctrl + P → ファイル名で検索。
  • コマンドパレット: Cmd/Ctrl + Shift + P → すべての機能を検索。
  • ターミナル: Ctrl + ` で表示/非表示。
  • 分割表示: Cmd/Ctrl + \ またはエディタ右上の分割ボタン。
  • 設定: Cmd/Ctrl + , でSettingsを開く。
マルチカーソル

同時編集のコツ

  • 縦に追加: Alt + クリック（MacはOption）。
  • 次の一致を追加: Cmd/Ctrl + D。
  • 全部まとめて選択: Cmd/Ctrl + Shift + L。
  • 矩形選択: Alt/Option + ドラッグ。
見つける/移動する

検索・置換

  • 現在のファイル内検索: Cmd/Ctrl + F。
  • 置換: Cmd/Ctrl + H。
  • プロジェクト全体検索: Cmd/Ctrl + Shift + F。
  • シンボルジャンプ: Cmd/Ctrl + Shift + O（ファイル内）/ T（ワークスペース）。
ショートカット

Mac / Windows 対応表

やりたいこと Mac Windows

ファイルを開く Cmd + P Ctrl + P
統合ターミナル Ctrl + ` Ctrl + `
サイドバー表示切替 Cmd + B Ctrl + B
分割エディタ Cmd + \ Ctrl + \
行へ移動 Ctrl + G Ctrl + G
コメントアウト Cmd + / Ctrl + /

Macでは基本的に Ctrl → Cmd に読み替えればOK。AltはOptionに相当します。

セットアップ

最初にやる設定

  • 自動保存: Settingsで"Auto Save"を"afterDelay"に。
  • フォントと行間: Editor Font Family/Line Heightを好みに調整。
  • ミニマップ非表示: Settingsで"Minimap"をオフにするとスッキリ。
