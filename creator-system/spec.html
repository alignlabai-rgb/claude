モジュール構成

5つのコアモジュール

各モジュールは独立したOpenClaw Skillとして実装され、メッセージキューで非同期連携します。

1

topic-analyzer

ネタ文章を受け取り、ジャンル・ターゲット・難易度を分析してメタデータを付与する。

Input: raw_text: string
Output: topic_data: TopicMeta
2

video-pipeline

topic_data からスクリプト生成・画像生成・音声合成・動画合成を行う。

Input: topic_data: TopicMeta
Output: video: ContentArtifact
3

manga-pipeline

topic_data からコマ割り・キャラ生成・セリフ配置・ページ合成を行う。

Input: topic_data: TopicMeta
Output: manga: ContentArtifact
4

content-evaluator

生成コンテンツをスコアリングし、公開可否フラグと改善提案を返す。

Input: artifact: ContentArtifact
Output: result: EvalResult
5

monetization-manager

評価通過コンテンツを指定プラットフォームに投稿し、収益データを記録する。

Input: artifact: ContentArtifact, score: number
Output: publish: PublishResult
型定義

共通データ型

TopicMeta

type TopicMeta = { id: string // UUID raw_text: string // 元ネタ文章 genre: string // "comedy"|"horror"|... target_age: string // "teen"|"adult"|"all" difficulty: number // 1–5 keywords: string[] // タグ created_at: string // ISO 8601 }

ContentArtifact

type ContentArtifact = { id: string type: "video" | "manga" file_path: string // ストレージパス thumbnail: string // サムネイルURL duration_sec: number// 動画の場合 page_count: number // 漫画の場合 topic_id: string // TopicMeta.id created_at: string }

EvalResult

type EvalResult = { artifact_id: string score: number // 0–100 passed: boolean // score >= 85 breakdown: { quality: number // 映像/画像品質 engagement: number// エンゲージメント予測 originality: number } suggestions: string[] }

PublishResult

type PublishResult = { artifact_id: string platforms: { name: string // "youtube"|"note"|... url: string published_at: string }[] revenue_est: number // 初期収益試算 (JPY) }
API エンドポイント

モジュール間インターフェース

POST /api/v1/topic-analyzer/analyze

ネタ文章を解析してTopicMetaを返す。

// Request { "raw_text": "ネタ文章をここに..." } // Response 200 { "topic_data": TopicMeta }
POST /api/v1/{video|manga}-pipeline/start

コンテンツ生成を開始する。非同期ジョブとして実行。

// Request { "topic_data": TopicMeta, "target_duration": 60 } // Response 202 { "job_id": "string", "status": "queued" }
POST /api/v1/evaluator/ingest

生成済みコンテンツをスコアリングキューに投入する。

// Request { "artifact": ContentArtifact } // Response 200 { "eval_result": EvalResult }
POST /api/v1/monetization/publish

評価通過コンテンツを指定プラットフォームに投稿する。

// Request { "artifact": ContentArtifact, "score": 92, "platforms": ["youtube", "note"] } // Response 200 { "publish_result": PublishResult }
エラーコード

共通エラー定義

コード意味対処
400リクエスト不正（必須フィールド欠損）ペイロードを確認する
422コンテンツ生成失敗（AIエラー）fallbackモデルに切り替え再試行
429APIレート制限超過exponential backoffで再試行
503外部AIサービス障害フォールバックサービスへ自動切替
