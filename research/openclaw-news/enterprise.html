1. エンタープライズ版の特徴

OpenClawはオープンソースですが、OpenClaw Foundationは企業向けのエンタープライズサポートとマネージドサービスを提供しています。

🔒 エンタープライズ専用機能
  • RBAC（ロールベースアクセス制御）： 管理者、オペレーター、閲覧者の3段階の権限管理
  • SSO連携： SAML 2.0 / OIDC対応。Okta、Azure AD、Google Workspaceと統合
  • 監査ログ： 全操作の詳細なログを90日間保持。SOC2準拠
  • データ暗号化： AES-256による保存データ暗号化、TLS 1.3による通信暗号化
  • SLA保証： 99.9%の稼働率保証（マネージドサービスプラン）

3. 導入アーキテクチャ

オンプレミスデプロイ

  • Kubernetes（推奨）： Helmチャートを提供。自動スケーリング・ローリングアップデート対応
  • Docker Compose： 小規模チーム向け。単一サーバーで完結
  • ベアメタル： 高性能が必要な場合。専用のインストールスクリプトを提供

クラウドデプロイ

  • AWS： Lightsail対応（公式パートナー）、ECS/EKSテンプレート
  • GCP： Cloud Run / GKEテンプレート
  • Azure： Container Apps / AKSテンプレート
  • Tencent Cloud： 中国リージョン向け公式サポート

4. コンプライアンス

📜 対応規制・認証
  • SOC 2 Type II： マネージドサービスプランで取得済み
  • GDPR： EU一般データ保護規則準拠。データ処理契約（DPA）を提供
  • HIPAA： 医療データ取扱い対応（BAA締結可能）
  • ISO 27001： 情報セキュリティマネジメントシステム認証
