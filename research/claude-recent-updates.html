🤖 Claude Code — 完全機能ガイド（2026年4月版）

Claude Code はターミナルから動くエージェント型コーディングアシスタント。 コードを読み・書き・実行・デバッグ・GitにPushまでを自律でこなす。 2026年に入り機能が急速に拡充し、「開発者のためのオペレーティングシステム」と呼ばれ始めている。

インストールとセットアップ

npm install -g @anthropic-ai/claude-code
claude-code --version   # バージョン確認
claude-code             # 起動

📋 主要機能一覧（2026年4月10日時点）

機能バージョン概要使い方・ポイント
/powerup コマンド NEW v2.1.90（4/1） インタラクティブなインターミナルチュートリアル。アニメーション付きデモで機能を学習 Claude Codeを起動後に /powerup と入力するだけ。デモにも使えるインタラクティブ教材。
Computer Use（CLI） NEW v2.1.86+ ターミナルからGUIアプリを操作。クリック・タイピング・スクロール・画面確認まで自律実行 操作優先順位: コネクター → ブラウザ → 画面コントロール。機密データには現在は非推奨。
Auto Mode（自動実行） v2.1.70+（3/24） 承認なしで連続タスクを自動実行。危険操作はブロックリストで制御 --auto フラグで起動。PermissionDenied hook で拒否後の {retry: true} リトライが可能。
Remote Control（リモート接続） Research Preview 長時間実行セッションをウェブ・デスクトップアプリから監視・操作できる 「実行中のセッションに後から繋いで状況確認」ができる。外出先からの管理に有効。
フリッカーフリーレンダリング v2.1.89（4月上旬） プロンプト入力が画面下部に固定され、ちらつきが完全消滅 CLAUDE_CODE_NO_FLICKER=1（古い環境向け旧式にフォールバック）
@ mention（サブエージェント名） v2.1.90（4/1） @ メンション入力時にサブエージェント名が候補表示されるように マルチエージェント構成でのエージェント指定が視覚的にわかりやすくなった。
MCP 強化（大容量ペイロード） v2.1.88+ ツールのresultサイズ上限が最大50万文字まで（per-tool override） 大規模DBスキーマや長いスプレッドシートもMCP経由でそのまま渡せる。
Plugin bin/ ディレクトリ対応 v2.1.87+ Pluginが bin/ に実行ファイルを置くと、アクティブ時に自動でPATHに追加される Pluginと一緒に外部ツールを配布できる。CLI拡張をバンドルしやすくなった。
GCP Vertex AI 統合ウィザード v2.1.90 ログインフロー内にGCP Vertex AI設定ウィザードが新設。 Anthropic API ではなく GCP 経由で使いたい企業が設定しやすくなった。
Windows 対応（Computer Use） 4/3追加 macOS限定だったComputer UseがWindowsにも対応 Pro/Max サブスクライバー向け。アプリ操作・ブラウザ・ファイル管理を Windows 上で実行。

Claude Code の典型的なワークフロー

# 1. プロジェクトディレクトリに移動
cd my-project

# 2. Claude Code 起動
claude-code

# 3. タスクを自然言語で指示
> "このリポジトリのバグを調査してください。テストが失敗している原因を特定して、
>  修正するPRブランチを作成し、変更内容をコミットしてください。"

# → Claude は自律的に:
#   1. ファイルツリーを読む
#   2. テストログを確認
#   3. 関連コードを特定
#   4. 修正を実装
#   5. git checkout -b fix/xxx でブランチ作成
#   6. git commit → git push まで完了

Claude Cowork とは（デモ前の理解として）

Claude Code の「コーディング能力」を知識ワーク全般に拡張した新プロダクト。

  • ドキュメント作成・データ分析・リサーチ・メール整理に対応
