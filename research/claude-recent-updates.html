2026年2月 — モデル刷新・Cowork登場

2/5 Claude Opus 4.6 リリース モデル注目

現行最高性能モデル。コーディング・計画・デバッグ・財務分析が大幅強化。

  • コンテキストウィンドウ 100万トークン（β）対応 — 企業文書ライブラリ丸ごと処理可能
  • タスク完了時間軸: 最大14.5時間の長期エージェント実行に対応
  • Message Batches API で max_tokens 上限を 300K に拡大
2/5 Claude Sonnet 4.6 リリース モデル

速度とコスパのバランスが最良のモデル。エージェント検索性能が向上しつつトークン消費を削減。1M トークンコンテキスト(β)も利用可。

2月初旬 Claude Cowork リサーチプレビュー開始 プロダクト注目

Claude Code のコーディング能力を「知識ワーク全般」に拡張した新プロダクト。

  • ドキュメント作成・データ分析・リサーチ・メール整理などに対応
  • Skill・Plugin・Connector を一元管理する「Customize」セクションを搭載
  • 定期・スポットのタスクスケジューリング機能
2月 Cowork Marketplace ローンチ プロダクト

プロフェッショナルワークフロー向けプラグインを集約したマーケットプレイス。Team・Enterprise プランで管理者コントロールも提供。

2月 Claude for PowerPoint（Add-in）プロダクト

Microsoft PowerPoint の Add-in として Claude が利用可能に。スライド文章の生成・改善・要約をPowerPoint内で直接実行。

2026年3月 — Computer Use・API強化・Marketplace

3/6 Claude Marketplace ローンチ プロダクト注目

サードパーティのClaudeベースサービスを統合するエンタープライズ向けマーケットプレイス。

  • Asana・Box・Canva・Figma・Hex・monday.com・Slack など主要サービスが参加
  • Claude のウィンドウを離れずに外部アプリのUIを操作できる MCP Apps 対応
3/24 Computer Use — macOS プレビュー公開 機能注目

Claude が実際にあなたのコンピュータを操作してタスクを完了する機能。Pro/Max サブスクライバー向けに macOS で先行公開。

  • 操作優先順位: コネクター → ブラウザ → 画面コントロール の順でフォールバック
  • 新しいアプリへのアクセス前に必ず許可を求める
  • Cowork の Dispatch 機能と連携: スマホで指示→デスクトップで実行→完了通知
  • 機密データへの利用はまだ非推奨（早期プレビュー段階）
3/24 Claude Code — Auto Mode Claude Code

Claude Code に自動実行モードが追加。承認なしで連続タスクを実行できる。

3月中旬 API: コード実行ツール・MCP コネクター・Files API・プロンプトキャッシュ API/開発注目

開発者向けに4つの新機能を同時リリース。

  • Code Execution Tool: Claude がコードを実行してデータ分析・計算を直接実施
  • MCP Connector: 外部システムへの接続を標準化（Notion・Google Drive 等）
  • Files API: セッションをまたいでファイルを効率的に保存・参照
  • Prompt Caching: 最大1時間のキャッシュでコストを大幅削減
3月中旬 Tool Search Tool（ツール動的探索）API/開発

全ツール定義を事前にロードせず、必要時に Claude が動的にツールを探索する仕組み。

  • defer_loading: true でオンデマンドに切り替え
  • トークン使用量 85%削減（122,800→191,300トークン保持）
  • 大規模ツールセットを持つエージェントに特に有効
3/26 Claude Mythos（次世代モデル）存在がリーク モデル未公式

Anthropic が社内でテスト中の次世代モデルがデータリークで判明。Fortune 誌が報道。

  • コーディング・学術推論・サイバーセキュリティで「段階的な能力向上」と評価
  • コードネーム Capybara: Opus を超える新ティアのモデルとされる
  • 早期アクセス顧客向けにトライアル中（2026年3月時点）
3/31 Claude Code ソースコード誤公開インシデント Claude Codeインシデント

デバッグファイルの設定ミスにより、Claude Code の全ソースコードが npm レジストリに一時的に公開された。

  • Anthropic は迅速に削除・対応
  • 意図的なリークか事故かで議論になったが公式は「ミス」と回答

2026年4月（上旬）— Windows対応・料金改定・Claude Code強化

4/3 Computer Use — Windows 対応追加 機能

macOS 限定だった Computer Use が Windows にも対応。Cowork と Claude Code の両方で利用可能に。

  • Pro/Max サブスクライバー向け
  • アプリ操作・ブラウザ・ファイル管理を Windows 上で実行
4/4 Claude Code — サードパーティツール課金ポリシー変更 Claude Code注目

OpenClaw など外部ハーネスからの Claude Code 利用が、サブスクリプション枠内では使えなくなる。

  • 2026年4月4日正午（太平洋時間）以降に適用
  • サードパーティ経由の使用は 従量課金（pay-as-you-go）に切り替え
  • claude.ai・公式IDE拡張からの利用はサブスクリプション枠内で継続
4月上旬 Claude Code — UX・安定性アップデート Claude Code

ユーザー体験と安定性の大幅改善。

  • フリッカーフリー alt-screen レンダリング（CLAUDE_CODE_NO_FLICKER=1 で有効化）
  • @ mention にサブエージェント名が表示されるように
  • PermissionDenied hook 追加: Auto Mode の拒否後に {retry: true} でリトライ可
  • インタラクティブな /powerup レッスン機能
  • 長時間セッションの安定性改善・Windows/音声問題の修正
