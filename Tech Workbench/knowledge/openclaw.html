概念: "Agentic AI" とは何か？

従来のLLM（ChatGPTなど）は「聞かれたら答える」受動的な存在でした。OpenClawなどのAgentic AI（自律エージェント）は、目的を与えられると、それを達成するために自ら行動計画を立て、実行します。

Self-Hosted クラウドではなく、あなたのローカルPCやVPS上で動作します。データはすべて手元にあり、外部に漏れません。
Action-Oriented 「コードを書いて」と言うと、チャット画面にコードを表示するのではなく、実際にファイルを作成し、テストを実行し、エラーがあれば修正まで行います。

コア・アーキテクチャ

OpenClawの能力を支える3つの柱。

  • Full System Access (Deep Integration)
    OSレベルの権限を持ちます。ファイルシステム、ターミナル、ブラウザ、インストールされた全アプリを操作可能です。これにより「環境構築」や「複雑なリサーチ」を完遂できます。
  • Persistent Memory (永続記憶)
    セッションが終わっても記憶はリセットされません。「以前話したプロジェクトAの件だけど」と言えば通じます。あなたの好み、癖、プロジェクトの文脈を永遠に学習し続けます。
  • Recursive Improvement (自己改善)
    タスク遂行に必要なツールがなければ、自らスクリプトを書いてインストールしたり、新しいスキルを習得しようとします。

Vibe Coding における役割

あなた（Human）と、Gemini（Architect）、そして OpenClaw（Executor）のトリオ編成。

Gemini (The Brain)

「このアプリの設計はどうすべきか？」「どういうUIがいいか？」といった、抽象度の高い思考と設計を担当。

OpenClaw (The Hands)

「設計図通りに全ファイルを生成して」「依存ライブラリを全部入れてテスト通るまでやっておいて」という、具体的かつ手数の多い作業を自律実行。

⚠️ DANGER: With Great Power...

OpenClawは「あなたのPCを自由に操作できる他者」と同義です。リスクを理解してください。

  • 破壊的操作のリスク: 強力な権限を持つため、誤った判断で重要なファイルを削除したり、システム設定を破壊する可能性があります。
  • Prompt Injection: 悪意あるWebサイトをOpenClawに閲覧させた場合、そのサイト内の指示に従ってPC内部の情報を外部へ送信してしまうリスクがあります。
  • Sandboxing: 運用時は必ずDockerなどのサンドボックス環境内や、権限を制限したユーザーアカウントで実行することを強く推奨します。

「便利な右腕」は、使い方を誤れば「最強のセキュリティホール」になります。

多様なインターフェース

ターミナルだけでなく、日常使うチャットアプリが「司令室」になります。

  • WhatsApp / Telegram / Discord: 外出先からスマホで「サーバーの様子どう？」と聞けば、ログを確認して返信してくれます。
  • Slack: チームメンバーの一員としてチャンネルに参加させることも可能。
