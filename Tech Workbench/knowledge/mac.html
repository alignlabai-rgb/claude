よくあるシーン

操作フローで理解する

アプリをまとめて閉じたい
▶
  • Cmd + Tabでアプリ一覧 → 対象を選択。
  • Cmd + Qでアプリ終了。ウィンドウだけ閉じるならCmd + W。
  • タブやウィンドウを一気に閉じるならOption + Cmd + W。
ファイルを切り取り移動したい
▶
  • Cmd + Cでコピー。
  • 移動先でCmd + Option + V → 実質「切り取り+貼り付け」。
  • 親フォルダへ戻る: Cmd + ↑。
スクショを撮り分けたい
▶
  • 範囲: Cmd + Shift + 4。
  • UI付きメニュー/録画: Cmd + Shift + 5（保存先も変更可）。
  • 全画面: Cmd + Shift + 3。
強制終了したい
▶
  • Option + Cmd + Escで強制終了ダイアログ。
  • 重い場合はSpotlight (CmdSpace) →「アクティビティモニタ」。
トラックパッドと配列

ジェスチャとキーの違い

  • 右クリック: 2本指クリック（またはControl + クリック）。
  • Mission Control: 3本指上スワイプ。デスクトップ切替は3本指左右。
  • デスクトップ表示: 親指+3本指を広げるジェスチャ。
  • Commandはスペースキーの左右、OptionはAlt相当、Controlは右クリック補助。
  • Deleteはバックスペース。Forward Deleteは Fn + Delete。

「CmdをCtrlの代わりに押す」クセをつけると一気に慣れます。

Finder

ファイル操作の基本

  • 開く: Enter（Return）。名前変更もEnterで編集モード。
  • 新規フォルダ: Shift + Cmd + N。
  • パスをコピー: Control + クリック → 情報をコピー、またはTerminalにドラッグ。
  • サイドバー非表示/表示: Option + Cmd + S。
  • 隠しファイル表示: Cmd + Shift + .
ウィンドウ

並べて・切り替えて使う

  • アプリ切替: Cmd + Tab。同一アプリ内のウィンドウ巡回は Cmd + `。
  • フルスクリーン: Ctrl + Cmd + F または緑ボタン。
  • 左右分割: 緑ボタン長押しでSplit View。
  • デスクトップ切替: Ctrl + → / ← または3本指左右。
  • Dock表示/非表示: Cmd + Option + D。
ブラウザ

タブ操作

  • 新規タブ: Cmd + T。タブを閉じる: Cmd + W。
  • 閉じたタブを戻す: Shift + Cmd + T。
  • ページ内検索: Cmd + F。
  • プライベート/シークレット: Cmd + Shift + N。
  • アドレスバーへ: Cmd + L。
