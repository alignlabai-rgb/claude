Troubleshooting

Diagnosing stuck agents and errors.
Agent is looping indefinitely

エージェントが同じ手順（Lint修正 → 失敗 → Retry）を永遠に繰り返している。

Possible Cause

"Blindness": エラーログを見ていない可能性があります。または、修正方法が間違っていることに気づいていません。

Solution

一度停止し、Hintsを与えて再開してください。

> Stop > Hint: "The error is in utils.ts. You need to export the interface."

または、`--max-steps` を短く設定し、早めに失敗させます。

Error: EACCES / Permission Denied

Docker使用時、ファイルを書き込もうとすると権限エラーになる。

Possible Cause

Linuxコンテナ内のUser IDと、ホスト側のUser IDが一致していません。

Solution

Docker実行時にUIDを渡します。

docker run -u $(id -u):$(id -g) ...
Agent edits the wrong file

指示していないファイルを勝手に書き換えたり、存在しないライブラリをimportする。

Possible Cause

Context Overflow: コンテキストが溢れ、制約条件（Constraints）を忘れています。

Solution

MISSION Specを分割し、小さくします。

Before: "Refactor entire app" After: "Refactor only /components/Button"
Error: ENOTFOUND (Network)

エージェントが外部サイト（npm registryなど）に接続できない。

Possible Cause

社内プロキシやVPNが、CLIからの接続を遮断しています。

Solution

プロキシ設定を環境変数に追加してください。

export HTTP_PROXY=http://proxy:8080 export HTTPS_PROXY=http://proxy:8080
