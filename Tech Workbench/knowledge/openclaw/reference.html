CLI & Config Reference

Complete list of commands, flags, and environment variables.
Main Commands
Command Description
openclaw run <file> 指定された Spec ファイル (例: MISSION.md) を読み込み、タスクを開始します。
$ openclaw run ./tasks/refactor.md
openclaw repl 対話モードで起動します。簡単なタスクや、動作確認用。
$ openclaw repl
openclaw doctor 環境診断コマンド。Docker, Git, Node.js のバージョンや権限を確認します。
openclaw version インストールされているバージョンを表示します。
Options & Flags
Flag Effect
--sandbox Dockerコンテナ内で実行します（推奨）。カレントディレクトリのみマウントされます。 Default: false (Native execution)
--max-steps <number> 思考/行動ループの最大回数。無限ループ防止用。 Default: 50
--max-cost <usd> API使用料の上限（ドル）。これを超えると強制停止します。 Default: 10.0
--debug / -d 詳細ログ（Verbose）を表示します。思考プロセスが全て見えます。
--model <name> 使用するLLMモデルを指定します。 Default: gpt-4o
--model o1-preview
--require-approval 危険な操作（書き込み、コマンド実行）の前にユーザー承認を求めます。
--require-approval="write,exec"
Environment Variables
Variable Description
OPENAI_API_KEY [必須] OpenAIのAPIキー。
OPENCLAW_HOME ログやキャッシュを保存するディレクトリ。 Default: ~/.openclaw
EDITOR `openclaw edit` でSpecを開く際に使用するエディタ。 Default: code (VS Code)
