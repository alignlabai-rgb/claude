Anti-Patterns

Detailed Failure Analysis & Exorcism Commands
⚠️ 1. The "Blind Fix" Loop

エージェントが「エラー内容を見ずに、当てずっぽうで修正」を繰り返し、コードを破壊する現象。

Symptoms (症状) > test failed: 'User' not found
> Claude: "Fixing..." (User.tsを修正)
> test failed: 'User' not found
> Claude: "Fixing..." (User.tsを元に戻す)
> test failed...
✅ The Exorcism Command

思考停止したエージェントの頬を叩くコマンドです。「手を動かすな、ログを見ろ」と命じます。

$ claude "STOP WRITING CODE. You are looping.
1. Read the error log carefully.
2. Read `src/models/User.ts` (do not edit yet).
3. Explain WHY the error persists in 1 sentence.
4. Only then, propose a fix."
⚠️ 2. The "Hallucinated Library" Trap

存在しない関数や、古いバージョンのAPIを自信満々に使い、"Undefined" エラーを引き起こす現象。

Symptoms (症状) > TypeError: supabase.auth.admin.listUsers is not a function
✅ The Reality Check

エージェントの知識は「トレーニング時点（過去）」で止まっています。最新のライブラリ仕様を教える必要があります。

$ claude "It seems you are hallucinating the API.
Read `node_modules/@supabase/supabase-js/package.json` to check the version.
Then, search the web or read the newly generated `docs/supabase.md` to find the correct method name."
⚠️ 3. Context Poisoning

長時間作業していると、過去の「間違った試行錯誤の履歴」がコンテキストに残り、エージェントが賢くなくなる現象。

Symptoms (症状) - 指示に対する反応が遅い
- 「前にも言ったけど」という修正を繰り返す
- 単純なミスが増える
✅ The Reset Button

人間で言う「睡眠」が必要です。履歴を消去し、今の正解だけを再度読み込ませます。

$ /clear
$ claude "Resuming work. Read `.ai/CONTEXT.md` and `src/current-focus.ts` to refresh your memory."
