Capabilities Cheatsheet

Power User Commands & Shell Integration

CLI版Claude Codeの真価は、OSのシェル（Bash/Zsh/PowerShell）との連携にあります。
単体で使うだけでなく、パイプラインの一部として組み込むことで最強の自動化ツールになります。

File Operations & Piping

$ claude --print "Userモデルの定義を作成して" > User.ts
Direct Output
Claudeにファイル操作権限を与えず、結果を標準出力に吐かせ、シェル側でファイルにリダイレクトする安全な手法。
$ cat error.log | claude "このエラーログの最後10行を分析して原因を突き止めて"
Log Analysis
巨大なログファイルをパイプで流し込む。Claudeは標準入力をコンテキストとして読み込みます。
$ claude --init --path ./frontend
Sub-directory Focus
モノレポ構成の場合、ルートではなく特定のディレクトリ（frontendなど）をルートと見なして分析させることができます。

Mode Flags

$ claude --effort low "fix lint errors"
Quick Fix Mode
思考時間を短縮し、Sonnet主体で高速に単純作業をこなします。コスト削減に有効。
$ claude --verbose "why is this failing?"
Brain Dump
Claudeの「思考プロセス（Thinking）」を隠さずすべて表示させます。彼が何を迷っているのかをデバッグするのに必須。
