制作フロー

7ステップで完成動画へ

1

ネタ文章の受け取り & 前処理

別ラインから生成されたテキストを受け取り、ジャンル・ターゲット・トーンを自動分析。不適切コンテンツフィルタリングも同時実行。

入力: テキスト GPT-4o / Claude
2

動画スクリプト生成

ネタ文章をYouTube/ショート動画向けのスクリプトに変換。フック（冒頭掴み）・本編構成・CTA（行動喚起）を自動設計。字幕テキストも同時生成。

// スクリプト生成プロンプト例 const script = await llm.generate({ system: "あなたはYouTube動画脚本家です", prompt: `以下のネタを元に動画スクリプトを作成: - フック: 最初の3秒で視聴者を掴む - 本編: ${targetDuration}秒の構成 - CTA: チャンネル登録・いいね誘導 ネタ: ${sourceText}` });
3

ビジュアル素材生成

スクリプトの各シーンに対応する画像・映像をAIで生成。スタイルの一貫性を保ちながら、シーンごとに最適な構図・色調を自動選択。

Midjourney API DALL-E 3 RunwayML Gen-3 Kling AI
4

音声合成（ナレーション）

スクリプトをリアルな日本語音声に変換。複数の声質から自動選択し、感情・速度・イントネーションを最適化。BGM選択も自動。

ElevenLabs VOICEVOX CoeFont Suno AI (BGM)
5

字幕・テロップ生成

音声に同期した字幕を自動生成。強調ワードのハイライト・エフェクト・フォントスタイルも動画の雰囲気に合わせて自動調整。

Whisper (文字起こし) FFmpeg (字幕埋め込み)
6

動画編集・結合

映像・音声・字幕をFFmpegで自動合成。トランジション・カット編集・テロップアニメーションをルールベースで適用し、完成動画を生成。

# FFmpegによる自動合成例 ffmpeg -i video_scenes.mp4 \ -i narration.mp3 \ -i bgm.mp3 \ -filter_complex "[1:a]volume=1.0[v1]; [2:a]volume=0.15[v2]; [v1][v2]amix=inputs=2[aout]" \ -map 0:v -map "[aout]" \ -vf "subtitles=subs.srt" \ output_final.mp4
7

サムネイル生成

動画の内容から最も視聴クリックされやすいサムネイルを複数パターン生成。A/Bテスト用に3〜5パターンを自動作成し評価システムへ送付。

Canva API DALL-E 3 出力: 評価システムへ
