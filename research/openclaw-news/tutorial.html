チュートリアル1: 初めてのOpenClaw

初級

OpenClawをインストールし、最初のタスクを実行するまでの手順です。所要時間：約15分。

1

インストール

# npmでインストール（推奨）
npm install -g openclaw

# または直接ダウンロード
curl -fsSL https://get.openclaw.dev | bash

# インストール確認
openclaw --version
2

初期設定

# 対話型セットアップウィザードを起動
openclaw init

# LLMプロバイダーを選択（Claude推奨）
# APIキーを入力
# メッセージングアプリを選択（Signal推奨）
3

最初のタスク実行

# OpenClawを起動
openclaw start

# テストタスクを送信
openclaw send "今日の天気を調べて、傘が必要かどうか教えて"

# タスク実行状況を確認
openclaw status

チュートリアル2: カスタムSkillの作成

中級

独自のSkillを作成して、OpenClawの機能を拡張します。所要時間：約30分。

1

Skillテンプレートを生成

# 新しいSkillのテンプレートを作成
openclaw skill init daily-summary

# ディレクトリ構造を確認
ls daily-summary/
# → SKILL.md  tools/  tests/
2

SKILL.mdを編集

# daily-summary/SKILL.md の内容

---
name: daily-summary
version: 1.0.0
description: 毎朝のデイリーサマリーを自動生成
triggers:
  - schedule: "0 7 * * *"  # 毎朝7時に実行
dependencies:
  - email-manager
  - calendar-sync
---

# Daily Summary Skill

## 目的
毎朝、以下の情報をまとめたサマリーを生成して送信する：
- 未読メールの要約（重要度順）
- 今日のスケジュール
- 天気予報
- ニュースヘッドライン

## 手順
1. email-manager Skillで未読メールを取得
2. calendar-sync Skillで今日の予定を取得
3. Web検索で天気予報とニュースを取得
4. すべての情報を構造化してサマリーを生成
5. Signalメッセージとして送信
3

テスト実行

# Skillの検証
openclaw skill validate ./daily-summary

# ローカルテスト実行
openclaw skill test ./daily-summary

# ワークスペースにインストール
openclaw skill install ./daily-summary

チュートリアル3: メッセージング連携

中級

SignalやTelegramからOpenClawを操作する設定を行います。所要時間：約20分。

📱 対応メッセージングアプリ
  • Signal（推奨）： E2E暗号化。プライバシー最重視の場合
  • Telegram： Bot APIが充実。グループでの共有利用に便利
  • Discord： 開発チーム向け。チャンネル別にSkillを割り当て可能
  • WhatsApp： Business API経由。設定がやや複雑
1

Signal連携の設定

# signal-cliのインストール
openclaw setup messaging --provider signal

# 電話番号を登録
openclaw config set --signal-number "+81XXXXXXXXXX"

# 認証コードを入力
openclaw verify --provider signal --code 123456

チュートリアル4: ワークフロー自動化

上級

複数のSkillを連携させた自動化ワークフローを構築します。所要時間：約45分。

🔄 例：顧客問い合わせ自動対応ワークフロー
  • トリガー： 新しいメールの受信を検知
  • 分類： メール内容を分析し、カテゴリ（技術サポート/営業/一般）に分類
  • ナレッジ検索： 社内ドキュメントDBから関連情報を検索
  • 返信下書き生成： 検索結果を基に返信文を自動生成
  • 人間レビュー： 生成された下書きをSignalで通知し、承認を待つ
  • 送信： 承認後、メールを自動送信

チュートリアル5: API経由での操作

上級

REST APIを使って外部アプリケーションからOpenClawを操作します。

1

PythonからAPIを呼び出す

import requests

API_URL = "http://localhost:3457/api/v1"
HEADERS = {"Authorization": "Bearer YOUR_TOKEN"}

# タスクを送信
response = requests.post(
    f"{API_URL}/tasks",
    headers=HEADERS,
    json={
        "message": "来週の会議をリスケジュールして",
        "priority": "high"
    }
)

task_id = response.json()["task_id"]
print(f"Task created: {task_id}")

# 結果を取得
result = requests.get(
    f"{API_URL}/tasks/{task_id}",
    headers=HEADERS
)
print(result.json())
