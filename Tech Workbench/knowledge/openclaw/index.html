Paradigm Shift: Director vs Operator

Claude Code (Director)

人間が常に画面の前にいる
対話しながら進める (Interactive)
細かく指示を出す (Micro-Management)
ミスがあればその場で指摘する

OpenClaw (Operator)

人間は寝ているか、別の仕事をしている
最初に目的だけ渡して放置 (Fire & Forget)
結果だけを受け取る (Result-Oriented)
失敗したらログを見て環境を調整する
