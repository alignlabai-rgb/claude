The Grand Cookbook

20+ "Fire & Forget" Missions for the Autonomous Age

これらは「人間がやるべきではない退屈な仕事」のリストです。
全て MISSION.md にコピペして、OpenClawに投げてください。

🧹 Maintenance (掃除と整備)
1. The Linter Fixer Night Shift

プロジェクト内の全Lintエラーを修正する。ロジック変更なしで。

Objective: Fix all `npm run lint` errors. Constraint: No logic change.
2. Type Guardian Night Shift

TypeScriptの `any` 型を撲滅し、適切な型定義に書き換える。

Objective: Replace `any` with specific types in `/src`. Reference: `types.ts`
3. The Translater

i18n対応。日本語のハードコードを探し、`en.json` `ja.json` に抽出する。

Objective: Extract hardcoded strings to i18n keys.
4. Dead Code Hunter

使われていない関数、変数、インポートを検出し、削除する。

Objective: Run `ts-prune` and remove unused exports.
5. Dependency Updater Risk

npmライブラリを最新化し、壊れたテストを直す。

Objective: `npm update` & fix build errors.
6. Commentator

複雑な関数にJSDocコメントを付与する。

Objective: Add JSDoc to all public functions in `/utils`.
🔨 Creation & Expansion (生成)
7. Test Generator

カバレッジが低いファイルの単体テストを量産する。

Objective: Create Jest tests for `src/components/*`. Target: 100% coverage.
8. Storybook Maker

コンポーネントをスキャンし、Storybookファイルを自動生成する。

Objective: Create `*.stories.tsx` for all UI components.
9. E2E Scripter

主要なユーザーフロー（ログイン→購入）のPlaywrightテストを書く。

Objective: Write Playwright E2E for "Checkout Flow".
10. README Writer

コードを解析し、最新の実装に基づいたREADME.mdを生成する。

Objective: Update README.md with current API specs.
11. CSS Module Migrator

巨大なglobal.cssを、コンポーネントごとのCSS Modulesに分割する。

Objective: Split `style.css` into `*.module.css`.
12. Mock Data Factory

開発用のダミーデータJSONを大量に生成する。

Objective: Generate 1000 users in `faker.json`.
🧠 Analysis & Research (調査)
13. The Competitor Spy

指定した競合サイトを巡回し、価格表や機能をExcelにまとめる。

Objective: Scrape prices from URLs and save to CSV.
14. Error Log Analyst

数千行のログファイルから、エラーのパターンと発生頻度を集計する。

Objective: Analyze `server.log` and summarize errors.
15. Library Auditor

プロジェクト内のライブラリのライセンス（MIT, GPL）を一覧化する。

Objective: List licenses for all node_modules.
16. Performance Audit

Lighthouseを実行し、スコアが低い箇所の改善案をレポートする。

Objective: Run Lighthouse & propose fix for LCP < 2.5s.
17. Security Scanner

コード内のハードコードされたパスワードやAPIキーを探す。

Objective: Find hardcoded secrets/keys in repo.
🏗️ Infra & Ops (運用)
18. Dockerizer

既存のアプリをコンテナ化するためのDockerfileを作成する。

Objective: Create Dockerfile & docker-compose.yml.
19. CI/CD Fixer

失敗しているGitHub Actionsのワークフローを修正する。

Objective: Fix `.github/workflows/test.yml`.
20. Database Seeder

テスト環境のDBに初期データを流し込むスクリプトを書く。

Objective: Create `seed.ts` for PostgreSQL.
